# Data Notes

The data files in this directory are generated controlled numerical parameters
and synthetic evidence tables used for methodological illustration.

They are not real field data. No proprietary industrial field data, private
maintenance records, or confidential asset records are included.

Running `python simulation.py` from the repository root regenerates these files:

- `controlled_parameters.csv`
- `load_multipliers.csv`
- `source_weight_scenarios.csv`
